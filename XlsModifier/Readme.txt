XlsModifier V3.4

Anleitung

1. Eine Kopie von Excel2GED.xlsm machen (das Programm verändert die Datei, wenn man es benutzt, man muss immer eine unmodifizierte Datei nbehmen)
2. XlsModifier.exe starten
3. Nun die Kopie der Excel2GED.xlsm auswählen in dem Requester
4. Nun ein Excel-Sheet im "Familia Austria" Format auswählen
5. Das (allerdings recht langsame) Programm trägt nun die Daten in das xlsm File ein
6. Die modifizierte xlsm Datei in Excel laden
7. Den Zielort eintragen und GEDCOM generieren klicken
8. Fertig

Diese Version konvertiert nur die 30 ersten Zeilen, eine Vollversion ist gegen 15 EUR bei mir (tirionareonwe@gmail.com, Steffen Häuser)
erhältlich.
